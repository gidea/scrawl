Scrawl works with any CLI agent you already have installed — no configuration required. Install an agent, open Scrawl, and it's detected automatically. For writing workflows, agents with strong instruction-following and long-form generation capabilities (Claude Code, Gemini, Qwen Code) tend to produce the best results.

## Feature Support

Not all agents support the same features:

**Auto-approve**: Skip permission prompts for file operations. Supported by Autohand Code, Claude Code, Codex, Gemini, Qwen, Cursor, Kilocode, Rovo, and Mistral.

**Resume**: Continue from a previous session. Supported by Claude Code, Gemini, Qwen, Kilocode, Continue, and Pi.

**Initial prompt**: Pass a writing brief when starting the agent. Most agents support this.

## Adding an Agent

Agents are detected automatically when installed. Open Scrawl, go to Settings, and click **Refresh** to scan for newly installed CLIs.

If an agent isn't detected, make sure its CLI is in your PATH.
